# Definition

`CourseAccordionHeaderContent` is a component that show course section information in course section accordion

it support loading state and accept course section information to display

- [loading](#primary)
- [default](#secondary)

### Default

### Loading

## Usage

The component comes in two variants `loading` and `default`

## Inputs

Button has the following properties:

- `loading` (type boolean optional default value false) - If `true`, show loading skeleton instead of content and in this case, it is not necessary to transmit other arguments
- `name` (type string optional when loading set to false in other case required) - course section name.
- `lessonNum` (type same as name) - The number of lesson in section.
- `duration` (type number optional when loading set to false in other case required) - total course section lessons duration in minutes
